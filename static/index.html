Features

schedule
Accurate and Up to Date

We check for updates to the chain registry several times each hour, so you'll always have the most up-to-date information.

add_task
Peer and Seed Liveness

Never worry about connecting to an offline peer again. We frequently check for peer and seed liveness. By default, peer endpoints only return live peers.

cloud_sync
Coming Soon! Endpoint Liveness

Like peers and seeds, we plan to check for RPC, LCD, and GRPC endpoint liveness while providing additional info such as earliest block height.

science
Testnets!

Discover and query for information about testnets too!

speed
Blazing Fast

Written in Rust, the API is built for speed. Never worry about Github rate limits again.

extension
OpenAPI / Swagger Compatible

Quickly generate client code in a variety of programming languages.

Get Started Now

# Discover mainnets
curl -s https://chainregistry.xyz/v1/mainnet/chains
# Discover Testnets
curl -s https://chainregistry.xyz/v1/testnet/chains
# Get chain data for Cosmos Hub
curl -s https://chainregistry.xyz/v1/mainnet/cosmoshub
# View the Hub's live peers and seeds
curl -s https://chainregistry.xyz/v1/mainnet/cosmoshub/peers
# Insert live peers into config. If using Mac, install and use gnu-sed.
sed -i 's/\(persistent_peers = "\).*\("\)/\1'$(curl -s https://chainregistry.xyz/v1/mainnet/cosmoshub/peers/peer_string)'\2/' config.toml
# Insert live seeds into config. If using Mac, install and use gnu-sed.
sed -i 's/\(seeds = "\).*\("\)/\1'$(curl -s https://chainregistry.xyz/v1/mainnet/cosmoshub/peers/seed_string)'\2/' config.toml

Generate
